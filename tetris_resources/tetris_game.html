Control Info:

Use the Left and Right buttons from the left or Key Arrows <- -> from the keyboard to move the bricks.

Use Space button or Rotate button to rotate the brick.

Use Down Arrow to accelerate the fall.
If the pile reaches reach to the top of the screen the game is over.

Tetris is a classic game in which the player has to move/rotate bricks of different forms while trying to fill out the rows from one end to the other. Filling out the row will award points and further space to build on.

The game ends if the bricks pile reaches the top rows.
